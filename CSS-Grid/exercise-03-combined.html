3.2 Asymmetric Grid (Real Layout Pattern)

Concept: Most real layouts aren't square! Mix different column and row sizes.

3.4 Responsive Grid Pattern

Concept: Combine auto-fit with minmax for both columns and rows to create fully responsive grids.

3.6 12-Column Grid System (Like Bootstrap!)

Concept: Create a 12-column system where items can span multiple columns. This is the foundation of many CSS frameworks.

grid-template-columns: repeat(12, 1fr); /* 12 equal columns */ grid-template-rows: auto; /* Rows size to content */
