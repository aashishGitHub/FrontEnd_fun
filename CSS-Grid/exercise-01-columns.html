What are ::before and ::after?

::before and ::after are CSS Pseudo-elements. They create virtual elements that don't exist in your HTML but can be styled and positioned.

üí° Key Points:
  • ::before adds content before the element
  • ::after adds content after the element
  • Both require the content property (even if empty: content: "")
  • You can use both on the same element!
  • They're inline by default, but can be styled like any element
üè≠ Industry Usage:

YES, pseudo-elements are EXTREMELY common in industry! They're used for:

  • Icons & Decorative Elements: Adding icons, arrows, quotes without extra HTML
  • Tooltips & Labels: Adding text labels (like we did here)
  • Shapes & Graphics: Creating triangles, circles, complex shapes with pure CSS
  • Clearfix: Fixing float layouts (classic use case)
  • Breadcrumbs: Adding separators between items
  • Form Validation: Adding checkmarks or error icons
  • Card Designs: Adding decorative corners, shadows, overlays
  • Typography: Drop caps, quotes, decorative text

üí° Pro Tip: Major frameworks like Bootstrap, Tailwind CSS, and Material Design use pseudo-elements extensively. They're a fundamental part of modern CSS!

üìù Syntax:
.element::before {
  content: "Text here";
  color: red;
}

.element::after {
  content: "Text here";
  color: blue;
}

Note: You can also use :before and :after (single colon) for older browser support, but ::before and ::after (double colon) is the modern standard for pseudo-elements.
