Concept: Define exact pixel heights for each row, just like we did with columns.

2.2 Fractional Rows (fr units)

Concept: Use 'fr' units to divide vertical space proportionally. Perfect for header/content/footer layouts!

grid-template-rows: 80px 1fr 60px; /* Header | Flexible Content | Footer */
Header (80px fixed)
Content Area (takes remaining space)

Footer (60px fixed)

2.4 Minmax for Rows

Concept: minmax() ensures rows have a minimum height but can grow if needed. Best of both worlds!

grid-template-rows: minmax(100px, auto) minmax(150px, 1fr) minmax(80px, auto); /* Min height, but can grow */

üí° Pro Tip: minmax(100px, auto) = "At least 100px, but grow for content". minmax(100px, 1fr) = "At least 100px, but share remaining space".

2.6 Real-World Layout: Dashboard

Concept: Combine fixed and flexible rows to create a complete dashboard layout.
